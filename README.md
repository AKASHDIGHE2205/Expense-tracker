# 💰 Daily Expense Tracker App

A full-stack mobile expense tracking application built using React Native Expo, TypeScript, Node.js, Express.js, and MongoDB. The app helps users manage daily income and expenses with a clean and responsive mobile interface.

---

## 🚀 Project Overview

* Developed a secure expense tracking mobile application with authentication and transaction management.
* Built dashboard analytics to monitor balance, income, and expenses with Day/Week/Month/Year filters.
* Implemented income and expense tracking with category-wise management.
* Added transaction search, date filtering, and recent transaction history.
* Designed a responsive and user-friendly mobile UI for Android and iOS platforms.
* Developed scalable REST APIs and integrated MongoDB for efficient data management.
* Implemented efficient server-state management and API caching using TanStack Query.
* Built the application using TypeScript for scalable and maintainable code architecture.
